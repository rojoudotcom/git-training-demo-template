みんなで作る、みんなのレシピ集

🥗 シーザーサラダ

⏱ 調理時間: 15分 👤 2人前

材料

  • ロメインレタス … 1株（またはレタス1/2個）
  • クルトン … 適量
  • 粉チーズ … 大さじ2
  • ベーコン … 2枚

ドレッシングの材料

  • マヨネーズ … 大さじ2
  • レモン汁 … 小さじ1
  • にんにく（すりおろし） … 少々
  • 粉チーズ … 大さじ1
  • 塩・こしょう … 少々

作り方

  1. レタスを洗い、食べやすい大きさにちぎって水気をしっかり切る。
  2. ベーコンを細切りにし、フライパンでカリカリになるまで焼く。
  3. ドレッシングの材料をボウルで混ぜ合わせる。
  4. 大きなボウルにレタスを入れ、ドレッシングを回しかけてざっくり和える。
  5. お皿に盛り付け、クルトン・ベーコン・粉チーズをトッピングして完成。

ポイント

  • レタスの水気をしっかり切ると、ドレッシングがよく絡みます。
  • 温泉卵をのせるとさらにおいしくなります。
